### Prerequisites

- Node.js 18 or higher
- Java 21 or higher (for testing Minecraft functionality)
- Git

### Quick Setup

```bash
# Clone your fork
git clone https://github.com/YOUR-USERNAME/clicraft.git
cd clicraft

# Install dependencies
npm install

# Link globally for testing
npm link

# Verify installation
clicraft --version
```
